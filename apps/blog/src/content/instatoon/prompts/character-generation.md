# 캐릭터 생성 프롬프트

캐릭터 시트와 표정·자세 에셋을 생성할 때 사용하는 공통 형식입니다.

```text
[캐릭터의 고정 외형과 복장], [표정과 자세], 흰색 또는 단순한 배경,
[공통 그림체 프롬프트]
```

## 사용 규칙

- 외형과 복장은 `../characters/<character-slug>/profile.md`에서 가져옵니다.
- 가능하면 항상 동일한 캐릭터 시트와 참조 이미지를 사용합니다.
- 이미지 안에는 말풍선이나 글자를 생성하지 않습니다.
- 채택한 결과는 해당 캐릭터의 `assets/`에 저장합니다.
