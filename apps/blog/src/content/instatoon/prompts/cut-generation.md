# 컷 생성 프롬프트

시나리오의 개별 컷을 생성할 때 아래 형식을 복사해 사용합니다.

## 템플릿

```text
[등장 캐릭터와 고정 외형], [현재 표정과 행동], [카메라 구도],
[배경과 주요 소품], [공통 그림체 프롬프트]
```

## 컷별 기록 항목

- 시나리오와 컷 번호:
- 목적:
- 등장 캐릭터:
- 구도와 카메라:
- 표정과 행동:
- 배경과 소품:
- 사용한 참조 이미지:
- 모델과 생성 설정:
- 최종 프롬프트:
- 생성 결과 파일:

컷에만 필요한 변형은 해당 시나리오의 `index.md` 제작 메모에 남깁니다.
